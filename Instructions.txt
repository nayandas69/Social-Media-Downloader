### 🎥 **For Users**  

#### 💻 **Getting Started**  
1. **Install Python:**  
   Grab it from [here](https://www.python.org/downloads/) if you haven’t already! Don’t forget to check **"Add Python to PATH"**—super important.  

2. **Install the Downloader:**  
   Open your terminal or cmd, and type:  
   ```bash  
   pip install social-media-downloader  
   ```  
   OR grab the `.exe` from our [releases](https://github.com/nayandas69/Social-Media-Downloader/releases/latest).

3. **Run the Tool:**  
   ```bash  
   social-media-downloader  
   ```  
   Paste your link, and BOOM! The magic begins. 🎩✨  

---

#### 🔥 **What Can You Do?**  

- **Snag individual vids & posts**: Paste any URL, and watch the tool work its charm. 💖  
- **Batch Mode Boss**: Got tons of links? Dump them into a text file (one URL per line) and run:  
  ```bash  
  social-media-downloader -batch file_name.txt  
  ```  

---

#### 🛠️ **Custom Vibes**  

1. **Edit `config.json`:**  
   🔧 Your gateway to total control! Open the file and tweak:  
   - `download_directory`: Set your fave folder for downloads. 🎯  
   - `mp3_quality`: Choose between `128`, `192`, or `320`. Customize your sound vibes! 🎧  
   - `default_format`: Change format preferences.  

2. **Logs & History:**  
   - All the receipts for your downloads live in:  
     - **Logs**: `downloader.log` 📜  
     - **History**: `download_history.csv` 🗃️  
   Check them out if you need details.  

---

#### ✨ **Pro Tips**  

1. **Speed Hack:**  
   Install [FFmpeg](https://ffmpeg.org/download.html) for smoother downloads and merging (you’re welcome 😉).  

2. **Stuck Offline?**  
   Relax! The tool will wait for you to connect and retry. No panics, just vibes. 🌐✨

### 👨‍💻 **For Developers**  

#### 💡 **Wanna Hack on This Project?**  

- **Step 1:** Fork the repo from GitHub:  
  👉 [Social Media Downloader Repo](https://github.com/nayandas69/Social-Media-Downloader)  

- **Step 2:** Clone it locally:  
  ```bash  
  git clone https://github.com/nayandas69/Social-Media-Downloader.git  
  cd Social-Media-Downloader  
  ```  

- **Step 3:** Set up a virtual environment (optional but classy):  
  ```bash  
  python -m venv venv  
  source venv/bin/activate  # Linux/Mac  
  venv\Scripts\activate  # Windows  
  ```  

- **Step 4:** Install dependencies:  
  ```bash  
  pip install -r requirements.txt  
  ```  

- **Step 5:** Run the magic:  
  ```bash  
  python downloader.py  
  ```

### ⚡ **FAQs**  

1. **Q: What platforms are supported?**  
   A: YouTube, TikTok, Instagram, and Facebook. 💃  

2. **Q: My download isn’t working! What now?**  
   A:  
   - Check your internet. Offline? The tool will wait patiently for you to reconnect. 🌐  
   - Make sure the URL is valid.  
   - Still an issue? Peek at `downloader.log` or [raise an issue](https://github.com/nayandas69/Social-Media-Downloader/issues).  

3. **Q: Can I use this without Python?**  
   A: Absolutely! Download the `.exe` from our [releases page](https://github.com/nayandas69/Social-Media-Downloader/releases/latest).  

4. **Q: Is this tool legal?**  
   A: Only for **personal use**. Downloading copyrighted content without permission isn’t chill. Play it safe, fam. 🙏
