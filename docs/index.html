Social Media Downloader

A powerful, open-source CLI tool to download public content from YouTube, Instagram, TikTok, Facebook, X and more — fast, safe & easy. By Nayan Das

Source Code PyPI

What is this?

This is a command-line tool to download public content from popular social media platforms like YouTube, Instagram, TikTok, and Facebook. It allows you to download videos and audio in various formats quickly and easily. The tool is designed to be lightweight and user-friendly, making it accessible for everyone. It is open-source and free to use, ensuring that you can download content without any restrictions. The tool is built using Python and is compatible with both Windows and Linux operating systems. It is designed to be fast and efficient, allowing you to download multiple files at once (only from Instagram) without any issues. The tool also includes features like history logs and update checks, ensuring that you always have the latest version and can keep track of your downloads.

Features

  • Fast batch downloads
  • Lightweight & easy to use
  • Open-source and free to use
  • History logs & update checks
  • Download public content only
  • Choose formats (MP3, MP4, etc)
  • Cross-platform (Windows/Linux)

How to Use

  1. Install via Python: pip install social-media-downloader
  2. Run the command: social-media-downloader
  3. For .deb files, use: sudo dpkg -i social-media-downloader_amd64.deb
  4. After installing the .deb package, run the command: smd
  5. For .exe files, run: Social.Media.Downloader.exe
  6. For Linux pre-binaries, extract the smd-linux.tar.gz file and run the script ./smd
  7. Follow on-screen steps
  8. For additional support, refer to the Support section.
